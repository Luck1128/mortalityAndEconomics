# Mortality and Economics
DSC104 Final project on the interplay between GDP, mortality, and time

GDP data for this project is provided by [The World Bank](https://datahelpdesk.worldbank.org/knowledgebase/articles/906519-world-bank-country-and-lending-groups), and mortality data is provided by the [Institute for Health Metrics and Evaluation](https://vizhub.healthdata.org/gbd-results/). Due to licensing restrictions from both organizations, the data from these sources cannot be disrubted directly in this repository, hence making the notebook files not runnable without currating the data individually. For this reason, PDFs of the notebooks already ran are included in this repository as well.
